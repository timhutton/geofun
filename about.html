Welcome to our playground for geolocation-based fun. This server is an experimental platform for games held in the real world which use the location services of smartphones and tablets.

In lieu of a proper information page, we list some resources around the web which relate to this server and the games it hosts.

Tim's repository
The README file gives hints on what to do and how to get involved with this project. You can also browse (and work on) the client source code.
geofun-announce
A low-volume announcement mailing list for playtesters. If playing games in Cambridge, UK, appeals to you, please join this list for news of our upcoming games. To join, email geofun-announce+subscribe@googlegroups.com.

Just want to say hi? Email hello@geofun.org.uk.
